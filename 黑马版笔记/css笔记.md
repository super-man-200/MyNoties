# 19.定位

![定位](E:\笔记\HTMLimg\定位.png)

## 1.相对定位

![相对定位](E:\笔记\HTMLimg\相对定位.png)

如果left和right都有，以left为准

如果top和bottom都有，以top为准

## 2.绝对定位

![绝对定位](E:\笔记\HTMLimg\绝对定位.png)

一层一层往上查找父级，找不到或者父级没有定位，以浏览器为参照物进行定位

一般父级采用相对定位模式（子绝父相）
